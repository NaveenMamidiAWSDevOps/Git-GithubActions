# What is Git ?

Git is a **Distributed Version Control System (DVCS)** used to track changes in code and manage version history.

---

# Git Basic Initial Commands :

* 1. `touch` : To create file(s).
     => Ex : `touch file1`
     `touch file1 file2 file3`

* 2. `mkdir` : To create folder(s) / directory(ies).
     => Ex : `mkdir folder1`
     `mkdir dir1 dir2 dir3`

* 3. `rm -rf` : To delete file(s) / directory(ies) forcefully. ⚠️
     => Ex : `rm -rf folder1`
     `rm -rf file1`

* 4. `ls` : Quick file list.

* 5. `ls -l` : Detailed file information.

* 6. `ls -a` : Show hidden files.

* 7. `ls -lrth` : Long listing + reverse time order + human-readable sizes.

* 8. `ls -lt` : Newest files first.

* 9. `ls -ltr` : Oldest files first.

* 10. `ls -lh` : Long format with human-readable sizes.

* 11. `ls -alh` : All files + detailed + human-readable.

* 12. `ll` : Detailed + hidden files (alias, may not work in all systems).

* 13. `tree` : Displays full directory structure.

* 14. `ls -la` : Detailed view of all files, including hidden system and Git files.

* 15. `cd` : Change directory.

* 16. `cat` : Displays file contents in the terminal.
      => Ex : `cat file1`
