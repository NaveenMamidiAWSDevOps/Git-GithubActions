# Introduction
* What is GitHub?

GitHub is a cloud-based platform used to host Git repositories. It provides additional collaboration and automation features such as pull requests, issue tracking, and CI/CD workflows.

I used GitHub to manage branching workflows, perform code reviews, and automate deployments using GitHub Actions. It also integrates effectively with tools such as Jira and AWS.

# Roles and Responsibilities as a DevOps Engineer in GitHub

As a DevOps Engineer, GitHub plays a central role in source code management, collaboration, and deployment automation. My responsibilities include creating and managing repositories and organizations, setting up teams, granting appropriate access permissions, and maintaining structured branching workflows. I also configure GitHub Actions for CI/CD pipelines, support pull request and code review processes, and integrate GitHub with tools such as Jira, AWS, Dependabot, and security scanning solutions to ensure efficient and secure software delivery.
